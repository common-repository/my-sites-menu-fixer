== Description ==

Shows the "My Sites" menu in three colums and sorts the sites alphabetically.


== Installation ==

Using this plugin is very simple. All you have to do is:


1. Upload the `my-sites-menu-fixer` folder to the `/wp-content/plugins/` directory
1. Network Activate the plugin through the 'Plugins' menu in Network Admin

Thats it!

== Frequently Asked Questions ==



= Who is this plugin for? =



This plugin is for Wordpress Multisites with to many sites to fit in the "My Sites" menu
